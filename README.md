## GO 학습 동기
2025년 11월 4일 21시~ 22시에 개발자 커뮤니티에서 Go 언어의 유용함에 대해 들었고,
강연이 끝난 뒤 Go에 대해 더 알아보니 흥미로운 점이 많길래 자연스럽게 학습 의욕이 더 생겼다.

> Go 언어는 **범용 프로그래밍 언어**로서 다양한 분야의 프로젝트에 활용된다.
특히 **시스템 프로그래밍**과 **백엔드 개발**에서 두드러지며,
쉽고 예외 상황이 적은 문법과 **빠른 성능**을 제공한다.
>
> 2009년 발표된 이후, Go 언어는 10년 넘게 **일관되고 간결한 문법**이라는 기조를 유지해왔다.
이러한 특징 덕분에 **빠른 개발이 필요한 스타트업**과
**높은 성능이 요구되는 시스템 프로그래밍** 분야에서 그 사용이 꾸준히 증가하고 있다.
현재는 **암호화폐, 웹 서버, 데이터베이스, 백엔드 시스템** 등에서 활발히 활용된다.
>
> — 《 Tucker의 Go 언어 프로그래밍》 中

## 준비물
Go 언어를 Visual Studio Code에서 사용하는 방법은 간단하다.
1. **Extensions(확장)** 탭에 접속한다.
2. **Go**를 검색해 **Google에서 제공하는 Go 확장 프로그램**을 설치한다.

## 실행시 주의사항
Go 1.16 버전부터 **Go 모듈이 기본 방식**으로 변경되었다.
이전에는 모든 코드가 `$GOPATH/src` 아래 있어야 했지만,
이제는 **각 프로젝트가 모듈 단위로 관리**된다.
따라서 예제를 빌드하기 전에 반드시 `go mod init` 명령으로 모듈을 생성해야 한다.

(대충 `go build` 명령어 쳐서 실행 파일 만들어야 실행할 수 있다는 말.
물론 `go run main.go`처럼 **빌드 없이 바로 실행하는 방법**도 있지만,
실행 파일을 만들어 배포하거나 재사용하려면 `go build`가 필요하다.)
